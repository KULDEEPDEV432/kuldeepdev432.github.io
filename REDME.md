# 🔧 ReadyBoot - Create Bootable USB Drives Easily

**ReadyBoot** is a free and lightweight tool that lets you create bootable USB drives using ISO files. It works without installation and supports Windows 7, 10, and 11.

![ReadyBoot Banner](https://kuldeepdev432.github.io/readyboot-image.png)

---

## 🚀 Features

- ⚡ Fast USB boot creation
- 💽 Supports ISO files
- 🖥 Compatible with Windows 7/10/11
- 🧩 No installation required
- 🎨 Lightweight and clean UI

## 💡 How to Use

1. Launch the app.
2. Select your USB drive.
3. Choose an ISO file.
4. Click **Start** and wait.
